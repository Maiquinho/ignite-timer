# Ignite Timer 

Projeto desenvolvido no Bootcamp Ignite (trilha 2022) na [Rocketseat](https://www.rocketseat.com.br/) focado em estudar os seguintes conceitos do Reactjs:


- Styled Components
- React Hook Form
- Zod
- React Router DOM
- Context API
- Hooks
    - useEffect
    - useReducer
- Imutabilidade
    - Immer
